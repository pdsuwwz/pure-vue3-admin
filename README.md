A Vue 3.x Pure Admin Prototyping Template

📦 一个纯净的 Vue3 模板，吸收敏捷开发思想，搭建基于 Vite5 + Vue3 + TypeScript + Pinia@next 的业务项目框架，实现快速开发，开箱即用

## Features

模板已集成以下技术栈，可按需引用

- [x] Vite 5.x
- [x] Vue 3.x
- [x] TypeScript
- [x] Vue Router
- [x] Pinia (替代 Vuex)
- [x] Element Plus 2.x
- [x] Jest 29.x + Unit Test
- [x] Lint + Flat Eslint(v9) + Stylelint
- [x] Axios
- [x] Lodash
- [x] Echarts
- [x] i18n + vue-i18n-next
- [x] Modular
- [x] Alias Path
